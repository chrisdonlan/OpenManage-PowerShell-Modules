# Invoke-OMEJobRun

## SYNOPSIS
Run job now

## SYNTAX

```
Invoke-OMEJobRun [-JobId] <Int32> [-Wait] [[-WaitTime] <Int32>] [<CommonParameters>]
```

## DESCRIPTION
Run an existing job by Id

## EXAMPLES

### EXAMPLE 1
```
28991 | Invoke-OMEJobRun -Wait -Verbose
```

Run job
